# Animeko - Website

简体中文 / [English](README.md)

一个一站式弹幕动画平台 [Animeko](https://github.com/open-ani/animeko) 的[官方网站](https://myani.org)

## 开发

1. 克隆此仓库至你的电脑

2. 运行

```shell
pnpm install
```

1. 若要启动开发服务器，请运行

```shell
pnpm dev
```

1. 若要构建并运行预览服务器，请运行

```shell
pnpm build && pnpm preview
```

>[!IMPORTANT]
>对于 Windows 上出现的报错，请使用 PowerShell 7
